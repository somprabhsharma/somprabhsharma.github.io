Dynamic Doctor Catalogue

Website - JSP

Developed this project as Trainee in Tech Mahindra Limited. In this project patient can set appointment with doctors from the directory of doctors and according to the schedule set by doctors. Patient can also ask any questions to doctors using this portal.
Concepts: JSP, Servlets, Oracle, HTML5, CSS3, JS, JQuery, Google gson
